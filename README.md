## Personal Portfolio Website

A modern, responsive developer portfolio website built to showcase my projects, skills, and experience.
Designed with a clean UI, smooth animations, and optimized performance.

### Live Demo: https://dineth-wickramasinghe.netlify.app/

## Features
Modern and minimal UI design
Fully responsive (mobile, tablet & desktop)
Fast performance with Vite
Smooth navigation & animations
Project showcase with images and details
About, Skills, Projects & Contact sections
Deployed on Netlify
Tech Stack
### Frontend
React
Vite
Tailwind CSS
JavaScript (ES6+)

Netlify – Deployment
Git & GitHub – Version control
Figma / Canva – UI inspiration
